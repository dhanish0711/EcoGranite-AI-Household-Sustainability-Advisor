# EcoGranite-AI-Household-Sustainability-Advisor
EcoGranite is a glassmorphic AI sustainability advisor aligned with UN SDG 12. It features dynamic carbon calculators, a Net-Zero goal planner, a waste sorting conveyor game, and a secure proxy to IBM watsonx.ai REST APIs, calling the Granite-3-8b model for personalized energy/water audits and actionable habits tracking.
